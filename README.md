### 1. ข้อมูลที่เราเก็บรวบรวม

Valkalie Bot เก็บข้อมูลเฉพาะที่จำเป็นสำหรับการทำงานของฟีเจอร์ต่างๆ ดังนี้

#### ข้อมูลของเซิร์ฟเวอร์ (Guild Data)
| ข้อมูล | วัตถุประสงค์ |
|--------|-------------|
| Guild ID | ระบุเซิร์ฟเวอร์สำหรับบันทึกการตั้งค่า |
| Channel ID | กำหนดช่องทางสำหรับระบบ Welcome/Leave และ Report |
| การตั้งค่า Welcome/Leave | แสดงข้อความต้อนรับและลาออกของสมาชิก |
| การตั้งค่า Report System | กำหนดช่องทางรับ Report และ Admin Panel |
| Embed ที่สร้างไว้ | บันทึก Embed ที่ Admin สร้างผ่าน Embed Builder |

#### ข้อมูลของผู้ใช้ (User Data)
| ข้อมูล | วัตถุประสงค์ |
|--------|-------------|
| User ID | ระบุผู้ใช้สำหรับระบบสถิติและ Report |
| สถิติการใช้งาน (Stats) | จำนวนข้อความ, เวลาในช่อง Voice ต่อเซิร์ฟเวอร์ |
| ข้อมูล Report | เหตุผลและหลักฐานที่ส่งผ่านระบบ Report |

#### ข้อมูลที่ **ไม่ได้** เก็บ
- ❌ เนื้อหาข้อความ (Message Content) — ไม่มีการบันทึกถาวร
- ❌ รหัสผ่านหรือข้อมูลส่วนตัวอื่นๆ
- ❌ ข้อมูลการเงิน
- ❌ ที่อยู่ IP ของผู้ใช้

### 2. วิธีที่เราใช้ข้อมูล

ข้อมูลที่เก็บรวบรวมถูกใช้เพื่อ:

- ให้บริการฟีเจอร์ต่างๆ ของบอทได้อย่างถูกต้อง (Welcome/Leave, Report, Stats, Embed Builder)
- แสดงสถิติการใช้งานเซิร์ฟเวอร์และผู้ใช้ผ่าน Stats Card
- เก็บ Log การทำงานชั่วคราวเพื่อแก้ไขปัญหา (สูงสุด 200 รายการ ไม่บันทึกถาวร)

---

### 3. การแบ่งปันข้อมูล

เราจะ **ไม่** แบ่งปัน ขาย หรือถ่ายโอนข้อมูลของคุณให้บุคคลภายนอก ยกเว้นในกรณีดังต่อไปนี้:

- เมื่อได้รับความยินยอมจากเจ้าของข้อมูลโดยชัดแจ้ง
- เมื่อกฎหมายกำหนดให้ต้องเปิดเผย

---

### 4. การจัดเก็บและความปลอดภัยของข้อมูล

- ข้อมูลถูกจัดเก็บในรูปแบบไฟล์ฐานข้อมูล SQLite (`stats.db`, `embeds.db`) และ JSON บนเซิร์ฟเวอร์ที่ควบคุมโดย Developer
- เราดำเนินมาตรการที่เหมาะสมเพื่อป้องกันการเข้าถึงข้อมูลโดยไม่ได้รับอนุญาต
- อย่างไรก็ตาม ไม่มีระบบใดที่ปลอดภัย 100% เราไม่สามารถรับประกันความปลอดภัยสมบูรณ์แบบได้

---

### 5. สิทธิ์ของผู้ใช้

คุณมีสิทธิ์:

- **ขอดูข้อมูล** ที่เราเก็บไว้เกี่ยวกับคุณหรือเซิร์ฟเวอร์ของคุณ
- **ขอลบข้อมูล** ของคุณออกจากระบบ
- **ลบบอทออกจากเซิร์ฟเวอร์** เมื่อใดก็ได้ — ข้อมูลของเซิร์ฟเวอร์นั้นจะถูกหยุดการใช้งาน

ในการใช้สิทธิ์ดังกล่าว กรุณาติดต่อเราผ่านช่องทางที่ระบุด้านล่าง

---

### 6. การเปลี่ยนแปลงนโยบาย

เราอาจอัปเดตนโยบายนี้เป็นครั้งคราว การใช้งานบอทอย่างต่อเนื่องหลังจากมีการเปลี่ยนแปลง ถือว่าคุณยอมรับนโยบายที่อัปเดตแล้ว

### 3. การใช้งานที่อนุญาต

Valkalie Bot มีไว้สำหรับใช้ในเซิร์ฟเวอร์ Discord ที่ถูกกฎหมาย เพื่อวัตถุประสงค์ดังต่อไปนี้:
- ฟีเจอร์เพลง (Music) สำหรับการบันเทิงในชุมชน
- ระบบ Welcome/Leave เพื่อต้อนรับสมาชิก
- ระบบ Report เพื่อรายงานพฤติกรรมที่ไม่เหมาะสม
- สถิติการใช้งานเพื่อดูแลชุมชน
- Embed Builder เพื่อสร้างประกาศและเนื้อหาต่างๆ

คุณตกลงว่าจะ **ไม่**:

- ใช้บอทเพื่อสแปม, คุกคาม, หรือล่วงละเมิดผู้อื่น
- พยายาม Exploit, Hack, หรือทำให้บอททำงานผิดปกติ
- ใช้บอทเพื่อกระทำสิ่งที่ขัดต่อ [Discord Community Guidelines](https://discord.com/guidelines) หรือกฎหมายที่ใช้บังคับ

- ใช้ฟีเจอร์ Music เพื่อเล่นเนื้อหาที่ละเมิดลิขสิทธิ์โดยเจตนา
